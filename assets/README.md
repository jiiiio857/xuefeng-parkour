## 图片缩放说明

- 以 `player.png` 为例：如果原图是 32×40px，目标高度 36px → 缩放比 = 36/40 = 0.9 → 实际渲染宽度 = 32×0.9 ≈ 29px，高度 = 36px
- 碰撞检测使用缩放后的实际尺寸，与渲染完全一致
- 如果图片加载失败，自动使用彩色像素风占位图
